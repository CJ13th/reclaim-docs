# Get Started

The many configuration in the InApp SDKs can be overriden. For example, a different attestor can be used, or we can update app info that appears in the verification flow.

<Callout type="info">
  The Reclaim Protocol SDK for overrides is available upon request. Contact our team to discuss implementation details.
</Callout>
