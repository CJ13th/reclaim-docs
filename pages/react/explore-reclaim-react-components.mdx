# React components (Reclaim)
[Reclaim Components](https://www.npmjs.com/package/@reclaimprotocol/react-components?activeTab=readme) package is a versatile solution for simplifying the integration of Reclaim functionality into React applications.
It offers pre-built, configurable React utility components, making it easy to work with the Reclaim SDK and quickly start development projects.
These components abstract the complexities of Reclaim-specific code, providing customizable user interfaces that allow developers to focus on their core business logic.
Additionally, the package comes with minimal default styles, enabling developers to easily adapt the components to match their application's visual design.

## Components

### GenerateProof
The GenerateProof component is a utility component provided by the [Reclaim Components](https://www.npmjs.com/package/@reclaimprotocol/react-components?activeTab=readme) package.
It enables a seamless integration for proof generation for a particular [provider](https://docs.reclaimprotocol.org/providers) by providing a minimal interface.

#### Usage
To use the GenerateProof component, follow these steps:

<Steps>

#### Import

```tsx
	import { GenerateProof } from '@reclaimprotocol/react-components';
```

#### Render
Include the component in your application's JSX code, providing the necessary props:
```tsx
<GenerateProof
  appID='6d6c04eb-237b-4599-8797-94d48b0ac612'
  userID='dasq2easdase-asdq2e3'
  onProofSubmission={() => {
    // Implement logic for successful proof submission
  }}
  onProofSubmissionFailed={() => {
    // Implement logic for failed proof submission
  }}
/>
```
