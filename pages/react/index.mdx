## Overview

Use Reclaim client SDK directly in your client code and get verified user data from Reclaim.

![sequence](/assets/Reclaim_client_SDK_integration_group.svg)

## Prerequisites

- Node.js 18.0.0 or later

## Installation

```bash copy
npm i @reclaimprotocol/reclaim-client-sdk
```

## Initialize the SDK

```js copy
import ReclaimSDK from '@reclaimprotocol/reclaim-client-sdk' 

const reclaimSDK = new ReclaimSDK('<YOUR-RECLAIM-APP-ID>')
```
